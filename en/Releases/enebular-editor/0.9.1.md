# 0.9.1 (February 1st, 2019) {#0.9.1 (February 1st, 2019)}

## New {#New}

N/A

## Fixed {#Fixed}

- Fixed an issue which prevented an error message from being displayed when attempting to login even if the network is disconnected and the attempt fails.
