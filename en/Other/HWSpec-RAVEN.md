* User defined LED and Switch
* DAPLink interface

    * USB Virtual COM Port
    * CMSIS-DAP Debugger
* Four Multi-function I/O Connectors
    * Compatible with 3.3V sensor and actuators
    * Multiple peripheral functions( ADC/UART/I2C/PWM )
